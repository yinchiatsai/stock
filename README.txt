# 金雀庫存管理系統｜多檔案版本

## 檔案結構

goldenbird-inventory-system/
├── index.html
├── css/
│   └── style.css
└── js/
    └── app.js

## 使用方式

1. 建立一個資料夾，例如：goldenbird-inventory-system
2. 把 index.html 放在最外層
3. 建立 css 資料夾，把 style.css 放進去
4. 建立 js 資料夾，把 app.js 放進去
5. 用瀏覽器打開 index.html

## 重要路徑

index.html 裡面有兩行很重要：

<link rel="stylesheet" href="css/style.css" />
<script src="js/app.js"></script>

代表：
- CSS 在 css/style.css
- JavaScript 在 js/app.js

## 目前資料儲存

目前仍是 localStorage 測試版，只會存在目前這台裝置。

正式版要改成：
- Firebase Firestore：存庫存、叫貨、對應表
- Firebase Storage：存截圖
- OCR API：辨識截圖

### 要正式啟用 Google 登入

請開啟 `js/firebase-auth.js`，修改兩個地方：

#### 1. Firebase config

```js
const firebaseConfig = {
  apiKey: "你的 apiKey",
  authDomain: "你的 project.firebaseapp.com",
  projectId: "你的 projectId",
  appId: "你的 appId"
};
```

#### 2. 人員權限

```js
const USER_ROLES = {
  "boss@gmail.com": "boss",
  "qing@gmail.com": "qing",
  "process@gmail.com": "process",
  "designer@gmail.com": "staff"
};
```
